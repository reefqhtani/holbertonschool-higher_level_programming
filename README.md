# Holberton School Higher Level Programming

This repository contains all projects and exercises for the **Higher Level Programming** track at Holberton School. 

## About

This curriculum focuses on learning various high-level programming languages and concepts beyond the C programming language. The track covers:

- **Python** - Object-oriented programming, data structures, and web frameworks
- **JavaScript** - Front-end and back-end development
- **SQL** - Database management and querying
- **APIs** - RESTful services and web development
- And more...

## Structure

The repository is organized by programming language and project type. Each directory contains specific projects with their own README files and requirements.

## Technologies

- Python
- JavaScript
- SQL
- HTML/CSS
- Flask
- jQuery
- and more...
