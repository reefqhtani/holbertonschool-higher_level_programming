# Python More Data Structures

## Overview
This project explores advanced Python data structures including sets, dictionaries, and functional programming concepts like lambda functions, map, filter, and reduce.

## Learning Objectives
- Understand and use sets and their methods
- Understand and use dictionaries and their methods
- Know when to use sets vs lists vs dictionaries
- Understand lambda functions
- Use map, filter, and reduce functions
- Iterate over sets and dictionaries
